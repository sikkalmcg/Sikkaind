# **App Name**: Sikka Logistics Platform

## Core Features:

- Live Dashboard: Display real-time logistics data, including available vehicles, pending shipments, and active trips, through dynamically updating summary cards.
- Vehicle Management: Allow adding and managing vehicle details such as entry/exit timestamps, driver information, and plant location.
- Shipment Planning: Enable creation of unique shipment IDs, origin/destination details, material information, and shipment status tracking.
- Vehicle Assignment: Assign vehicles to shipments, map drivers, generate trip IDs, and manage assignment statuses.
- Shipment Tracking: Track vehicle number, shipment ID, trip ID, current location, current status and updated timestamp, which can be integrated with external APIs such as Google Maps or others
- Freight Management: Manage freight amounts, advance payments, balance calculations, and payment statuses.
- Admin Panel: Administer Vehicle, Plant, and Fuel Management features, as well as manage Sub-User access to various app modules using page-wise authorization.

## Style Guidelines:

- Primary color: Professional blue (#2962FF) to align with the requested theme. It evokes a sense of trust, stability, and professionalism, essential for logistics.
- Background color: Light blue (#E3F2FD) for a clean and modern look. It provides a soft, unobtrusive backdrop that makes the primary elements stand out.
- Accent color: Electric Blue (#7DF9FF) for CTAs and highlights. Being an analogous color of blue, this shade brings brightness and focus to interactive elements while still working within the theme.
- Body and headline font: 'Inter', a sans-serif font known for its readability and clean, modern aesthetic. Suitable for both headlines and body text, enhancing the user experience across the platform. Note: currently only Google Fonts are supported.
- Card-based layout for the dashboard and other modules to present information in an organized manner.
- Consistent and intuitive icons for navigation and actions throughout the application.
- Subtle loading indicators and transitions for a smooth user experience.
